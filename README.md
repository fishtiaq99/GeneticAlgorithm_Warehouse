# WarehouseStorageGA

Assignment 2 for Artificial Intelligence.

This project implements a Genetic Algorithm (GA) to optimize warehouse storage allocation while satisfying operational constraints such as:

- Weight capacity limits
- Fragile item protection
- Hazardous material isolation
- Temperature-controlled storage
- High-demand product accessibility
- Product compatibility rules

## Features

- Genetic Algorithm based optimization
- Fitness evaluation using penalty scoring
- Selection, crossover, and mutation operations
- Warehouse zone allocation system
- Excel export support for optimized storage plans

## Files

- `assignment2.py` → Main implementation
- `warehouse_storage_plan.xlsx` → Optimized storage output
- `A2_E_23i-0696_Fatima_Ishtiaq.docx` → Assignment report
